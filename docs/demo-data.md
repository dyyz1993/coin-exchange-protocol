## 演示页面功能

### 1. 系统概览（Dashboard）
- 📊 总账户数统计（12,458）
- 💰 总交易额展示（8,234,567 金币）
- 🔒 冻结账户数量（47）
- ✅ 活跃任务数量（128）

### 2. 账户管理
- 显示所有账户列表
- 展示账户余额和冻结金额
- 账户状态标识（正常/已冻结）
- 快速操作按钮（冻结/解冻）

### 3. 冻结管理
- 冻结申请列表
- 冻结记录查看
- 冻结状态统计
- 审核流程展示

### 4. 交易记录
- 所有交易历史
- 交易类型筛选（转账/充值/提现）
- 交易状态追踪
- 导出功能

## UI/UX 特性

### 响应式设计
- ✅ 支持桌面端（> 1024px）
- ✅ 支持平板端（768px - 1024px）
- ✅ 支持移动端（< 768px）

### 交互效果
- ✅ 平滑过渡动画
- ✅ 悬停效果
- ✅ 状态徽章动画
- ✅ 卡片阴影效果

### 视觉设计
- 🎨 渐变色导航栏

- 🎨 清晰的数据展示
- 🎨 现代化界面风格

## 截图建议

为 E2E 测试和文档准备截图时，建议：

1. **仪表盘截图**
   - 确保所有统计数据可见
   - 使用 1920x1080 分辨率

2. **账户管理截图**
   - 展示表格完整内容
   - 突出显示不同状态的账户

3. **冻结管理截图**
   - 显示冻结申请流程
   - 展示冻结记录详情

4. **交易记录截图**
   - 包含不同类型的交易
   - 展示不同状态的交易

## 启动服务

```bash
# 开发模式（支持热重载）
bun run dev

# 生产模式
bun run start
```
