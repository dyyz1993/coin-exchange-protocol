## 📝 经验总结

### 问题根源
1. **缺乏检查机制**：创建 Issue 前未强制搜索
2. **没有模板**：Issue 格式不统一，难以识别重复
3. **维护不及时**：长时间未清理导致累积

### 改进措施
1. **使用模板**：创建标准 Issue 模板
2. **定期清理**：每两周进行一次 Issue 审查
3. **自动化工具**：考虑使用 GitHub Bot 自动标记重复
